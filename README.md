# 陪练猫工具包

陪练猫工具包是一款面向内容创作者的桌面应用，支持短视频脚本、AI 朋友圈、AI 图文发布、模型管理和后台提示词同步。

## 主要功能

- 短视频脚本：根据选题生成脚本结构和分镜内容。
- AI 朋友圈：支持朋友圈改写、朋友圈生成、AI 配图和参考图生图。
- AI 图文发布：根据少儿英语/英语启蒙选题生成抖音图文卡片、标题、正文和话题。
- 后端管理：本机管理语言大模型和生图大模型。
- 提示词同步：网页后台维护提示词，终端应用一键更新后台提示词。

## 云后台

提示词后台地址：

<https://peilianmao001.com/admin>

网页后台只维护提示词。终端应用在“后端管理 -> 提示词管理”里点击“更新后台提示词”，即可把后台提示词同步到本机应用。

## 本地开发

```bash
npm install
npm run dev
```

## 打包安装包

```bash
npm run dist
```

Windows 安装包会生成在 `release` 目录。
